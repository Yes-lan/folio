# Portfolio statique

Ce dépôt contient un portfolio statique léger prêt à être personnalisé.

Fichiers importants :

- `index.html` — structure du site (sections : À propos, Stage, Projets, Certifications, Veille techno)
- `styles.css` — styles et thèmes clair/sombre
- `script.js` — petites interactions : bascule thème, menu mobile, année automatique
- `assets/avatar.svg` — image placeholder

Comment personnaliser :

1. Ouvre `index.html` et remplace les textes de démonstration par ton contenu (nom, bio, projets, etc.).
2. Ajoute des images et captures d'écran dans `assets/` et mets à jour les chemins dans `index.html`.
3. Pour l'hébergement gratuit, tu peux déployer sur GitHub Pages :

   - Crée un dépôt GitHub
   - Pousse les fichiers
   - Active GitHub Pages sur la branche `main` et le dossier `/`.

Améliorations possibles :

- Ajouter un formulaire de contact (Netlify Forms ou backend)
- Ajouter des pages dédiées pour chaque projet
- Intégrer un CMS léger (Netlify CMS, TinaCMS) pour éditer sans code

Si tu veux, je peux :

- Remplir le contenu si tu me donnes ta bio, infos de stage, détails projets, certifications et centres de veille.
- Générer une version en React/Vite ou un template prêt à déployer sur GitHub Pages.
